# Crypto Data Engine

Crypto Data Engine is a FastAPI-based backend system that collects cryptocurrency market data and exposes it through REST APIs. It supports OHLCV candlestick data, ticker prices, trades, and market metrics while using PostgreSQL for storage and Redis for caching.
